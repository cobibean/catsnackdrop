# How to Clone This Repo

Cloning means downloading a copy of this code to your computer so you can run it.

## Method 1: Using GitHub Desktop (Easiest for Beginners)

1. **Install GitHub Desktop:**
   - Go to [desktop.github.com](https://desktop.github.com/)
   - Download and install it

2. **Clone the repo:**
   - Open GitHub Desktop
   - Click "File" → "Clone Repository"
   - Go to the "URL" tab
   - Paste this URL: `https://github.com/cobibean/catsnackdrop`
   - Choose where to save it (like your Desktop)
   - Click "Clone"

3. **Open in Cursor:**
   - In GitHub Desktop, right-click the repo
   - Click "Open in Cursor" (or "Open in Visual Studio Code")

## Method 2: Using the Terminal (More Control)

1. **Open the terminal in Cursor:**
   - View → Terminal (or press `` Ctrl+` ``)

2. **Navigate to where you want the project:**
   ```bash
   cd ~/Desktop
   ```
   (This goes to your Desktop. You can use any folder you want.)

3. **Clone the repo:**
   ```bash
   git clone https://github.com/cobibean/catsnackdrop.git
   ```
   Copy and paste this entire line, then press Enter.

4. **Open the folder in Cursor:**
   - File → Open Folder
   - Navigate to the `catsnackdrop` folder you just cloned
   - Click "Open"

## Method 3: Download as ZIP (No Git Required)

1. **On GitHub:**
   - Go to [https://github.com/cobibean/catsnackdrop](https://github.com/cobibean/catsnackdrop)
   - Click the green "Code" button
   - Click "Download ZIP"

2. **Extract the ZIP:**
   - Find the downloaded file
   - Right-click → "Extract All" (Windows) or double-click (Mac)

3. **Open in Cursor:**
   - File → Open Folder
   - Select the extracted `catsnackdrop` folder

---

**Which method should you use?**
- **Method 1** if you want a visual tool
- **Method 2** if you want to learn Git commands
- **Method 3** if you just want the code and don't care about Git

All three methods work! Pick whichever feels easiest to you.

## What About Forking?

**Forking** creates your own copy of this repo on GitHub that you can modify and keep separate.

**When to fork:**
- You want to make changes and keep them on your GitHub
- You want to contribute back to the project
- You want your own version to experiment with

**How to fork:**
1. On GitHub, go to [https://github.com/cobibean/catsnackdrop](https://github.com/cobibean/catsnackdrop)
2. Click the "Fork" button (top right)
3. Choose your account
4. Wait for it to finish
5. Then clone YOUR fork (use the same methods above, but use your fork's URL instead)

**For beginners:** You probably just want to **clone** it to play the game. Forking is only needed if you want to modify it and keep your changes on GitHub.
